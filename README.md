Zerodu Framework
================

## 开发环境约定
 * product  生产环境
 * devel    开发环境
 * local    本地环境

## 目录结构
```
.
| (业务层)
├── application
│   ├── controllers
│   ├── (组件)
│   ├── hooks
│   ├── models
│   └── views
| (公共层)
├── common
│   ├── LDAP
│   ├── OSS
│   └── Paas
| (入口&配置层)
├── config
├── public
│   └── index.php
| (框架层)
├── system
└── vendor
```
